# 🌐 Subodh Kumar Agrahari - Portfolio Website

A modern, responsive **personal portfolio website** built using **React, TypeScript, and Tailwind CSS**.  
Designed to showcase projects, skills, and experience with a clean UI and smooth user experience.

---

## 🚀 Live Demo
👉 https://sk9-portfolio.vercel.app

---


## ✨ Features

- 📱 **Fully Responsive Design**  
  Works seamlessly across mobile, tablet, and desktop devices  

- 🌙 **Dark / Light Mode**  
  Toggle theme with saved user preference  

- ⚡ **Fast Performance**  
  Powered by Vite for lightning-fast builds  

- 🎯 **Smooth Animations**  
  - Typing animation in hero section  
  - Scroll reveal effects  
  - Smooth navigation  

- 🧭 **Sticky Navigation Bar**  
  Easy section navigation with smooth scrolling  

- 🎨 **Modern UI/UX**  
  Clean layout with gradients, hover effects, and minimal design  

- 🔍 **SEO Optimized**  
  Meta tags for better search engine visibility  

---

## 🧩 Sections Included

- **Hero** – Introduction with animation  
- **About** – Background and education  
- **Skills** – Interactive progress bars  
- **Projects** – Showcase of work  
- **Experience** – Work history  
- **Certifications** – Achievements  
- **Contact** – Contact form and links  

---

## 🛠️ Tech Stack

- ⚛️ React 18  
- 📘 TypeScript  
- 🎨 Tailwind CSS  
- ⚡ Vite

## 📂 Project Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/subodh-git77/my-portfolio.git
cd my-portfolio
```
### 2️⃣ Install dependencies:
```bash
npm install
```

### 3️⃣ Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`
